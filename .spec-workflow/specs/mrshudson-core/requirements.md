# MrsHudson Core - 需求文档

## 概述

**规格名称**: mrshudson-core
**描述**: MrsHudson AI管家的核心系统，包含用户认证、AI对话引擎(Kimi)、语音交互、MCP工具集成、Web端和Android端支持
**状态**: 已完成
**完成日期**: 2026-03-09

---

> **说明**: iOS 开发作为后续扩展计划，优先级较低。

---

## 用户故事

### US-001: 简化登录（Demo版）
**作为** Demo用户，
**我希望** 使用简单的admin账号快速登录，
**以便** 快速体验MrsHudson功能。

**验收标准**:
- [x] 预设admin/admin账号
- [x] 简单表单登录
- [x] JWT Token认证（Access Token + Refresh Token）
- [x] Token自动刷新机制
- [ ] 暂不需要注册功能
- [x] 密码BCrypt加密存储

**注**: 完整用户系统将作为V2功能开发

---

### US-002: 自然语言对话（Kimi驱动）
**作为** 用户，
**我希望** 能用自然语言与MrsHudson对话，
**以便** 像与真人管家一样交流。

**验收标准**:
- [x] 支持文字输入对话
- [x] Kimi AI理解用户意图（天气查询、日程安排、待办管理等）
- [x] 上下文保持（多轮对话）
- [x] 响应时间 < 2秒
- [x] 支持中文和英文
- [x] 支持Function Calling调用MCP工具

---

### US-003: 天气查询服务
**作为** 用户，
**我希望** 询问MrsHudson天气情况，
**以便** 了解当前和未来的天气。

**验收标准**:
- [x] 查询当前天气（温度、湿度、天气状况）
- [x] 查询未来7天预报
- [~] 主动推送恶劣天气提醒（需 FCM 推送启用）
- [x] 根据天气给出穿衣建议

---

### US-004: 日历事件管理
**作为** 用户，
**我希望** 通过对话管理我的日程，
**以便** 高效安排时间。

**验收标准**:

- [x] 事件提醒（开始前15分钟默认）
- [x] 事件分类（工作、个人、家庭）
- [~] 支持重复事件（后端支持，前端 UI 未完整实现）

### US-008: 语音交互
**作为** 用户，
**我希望** 能够通过语音与MrsHudson交流，
**以便** 在不方便打字的场景下也能使用。

**验收标准**:
- [x] 支持语音输入（语音识别）
  - Android: 使用 Android SpeechRecognizer API
- [x] 支持语音输出（语音合成）
  - Web: 已实现 TTS 播放
  - Android: 缺 TTS 播放 UI
- [ ] 支持唤醒词激活（可选："嘿，哈德森夫人"）
- [~] 支持多种音色选择（讯飞 TTS 支持，UI 未开放）
- [ ] 语音识别准确率 > 95%（中文普通话）
- [ ] 支持语音打断和纠错

**技术实现**:
- Android: SpeechRecognizer API（系统内置，无需第三方SDK）

---

### US-009: 多端应用支持
**作为** 用户，
**我希望** 在不同设备上使用MrsHudson，
**以便** 随时随地获得服务。

**验收标准**:
- [x] Web端（桌面浏览器）- Vue 3 + Element Plus
- [x] Android原生应用 - Jetpack Compose
  - [x] Kotlin + Jetpack Compose 架构
  - [x] MVVM + Repository 设计模式
  - [x] Room 本地数据库离线支持
  - [~] FCM 推送通知集成（框架已实现，需配置 Firebase）
  - [x] 语音输入（Android SpeechRecognizer）
- [x] 数据多端同步（后端已支持）
- [x] 统一的用户认证体系（JWT）
- [~] 消息推送（Android FCM，需配置 Firebase）
- [x] 离线模式支持（查看缓存数据）

**后续扩展**:
- iOS 原生应用（SwiftUI）
- 微信小程序
- 支付宝小程序
- Mobile Web（H5）

## 非功能性需求

### 性能
- 页面加载时间 < 3秒
- API响应时间 < 500ms（简单查询）
- AI对话响应时间 < 2秒
- 支持1000并发用户

### 安全
- 所有API需认证
- 敏感数据加密
- 防止SQL注入
- XSS防护
- CSRF防护

### 可用性
- 系统可用性 99.9%
- 支持主流浏览器（Chrome, Firefox, Safari, Edge）
- 响应式设计（支持移动端）
